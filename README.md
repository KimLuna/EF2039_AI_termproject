# EF2039_AI_termproject

## Environment & Prerequisites
This project requires a specific version of Python to run the Spleeter library correctly.

* **Python Version**: **3.9 or 3.10** (Recommended)
  * *Warning: This project may not work on Python 3.11 or higher due to `tensorflow` and `spleeter` compatibility issues.*
* **Required Tools**:
  * **FFmpeg**: This library is required for audio processing.
    * Ubuntu: `sudo apt-get install ffmpeg`
    * Windows: Install via Chocolatey or download executable.
